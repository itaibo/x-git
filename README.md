# @itaibo/x-git

Git commands to use with x [@type-x/cli](https://www.npmjs.com/package/@type-x/cli)

## Install

```sh
# If you don't have x installed yet
npm i -g @type-x/cli@latest

x add @itaibo/x-git
```

## Commands

```sh
# Create new feature branch
x new

# Pull from origin
x pull

# Push to origin
x push

```

## Development

```sh
pnpm install
pnpm build
x run . pull
x run . push
```
